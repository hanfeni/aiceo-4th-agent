**① 도메인 색인** (텍스트 → OpenSearch) — */index-lab*

- **색인된 데이터 확인** (인덱스 보기 모달 — 50개 페이지네이션·raw JSON·임베딩 1536-d)
- (역할) 색인은 검색·RAG의 **준비 단계** — 여기서 만든 인덱스를 ④검색 실습이 소비

**② 데이터 적재** (표 → SQLite) — */data-load*
- **도메인별 원본 보기** (적재 전 GitHub 원본 CSV 앞 20행 미리보기)
- **파라미터 설정 후 적재** (CSV→SQLite 테이블 적재, SSE 진행로그 / 도메인별 초기화)
- **적재된 데이터 확인** (적재 현황·테이블 스키마)
- (역할) 적재는 Text-to-SQL의 **준비 단계** — 여기서 만든 테이블을 ④검색 실습이 소비
- (대비 포인트) 색인=의미검색용(동의어·문맥) ↔ 적재=정확 쿼리용(집계·조건) — 데이터 형태로 갈림

**③ 메타 라벨링 실습** (LLM이 데이터에 메타를 붙인다 → 검색 품질의 4번째 축) — */meta-lab*
