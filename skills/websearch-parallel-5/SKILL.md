# 웹검색 병렬 실행 5개

## When to use
- 웹검색이 필요하고, 가능한 한 빠르게 여러 결과를 모아야 할 때 사용한다.
- 하나의 검색 질의만으로 충분하지 않고, 다양한 키워드 조합이나 관점이 필요할 때 사용한다.
- 검색 도구를 여러 개 동시에 호출해 지연 시간을 줄이고 싶을 때 사용한다.
- 최신 정보, 교차 검증, 여러 출처 비교가 중요한 경우에 사용한다.

## How
1. 사용자의 검색 목적을 5개의 서로 다른 검색 질의로 분해한다.
2. 각 질의는 중복을 줄이고 관점을 다르게 구성한다.
   - 핵심 키워드 중심
   - 동의어/약어 포함
   - 공식 문서/공식 사이트 중심
   - 뉴스/블로그/커뮤니티 중심
   - 비교/리뷰/사례 중심
3. 웹검색 도구 5개를 동시에 실행한다.
4. 결과를 수집한 뒤 중복 결과를 제거하고 신뢰도와 관련도로 정리한다.
5. 서로 다른 출처의 공통점과 차이점을 확인해 요약한다.
6. 결과가 부족하면 추가 검색을 하되, 먼저 5개 병렬 실행을 기본으로 유지한다.
7. 검색 결과를 답변에 반영할 때는 출처가 명확한 정보와 추정 정보를 구분한다.

### 권장 검색 질의 예시
- 질의 1: 기본 핵심 키워드
- 질의 2: 영문/원문 키워드
- 질의 3: 공식 문서/공식 사이트 키워드
- 질의 4: 최신 뉴스/업데이트 키워드
- 질의 5: 비교/문제 해결/사례 키워드
