# accounting-analysis-framework

## When to use
회계 분석, 재무제표 검토, 회계정책 해석, 분식 가능성 점검, 손익·재무상태·현금흐름 간 정합성 확인이 필요할 때 사용합니다.

## How
1. 분석 대상의 목적을 먼저 파악한다.
2. 재무제표 핵심 항목(손익계산서, 재무상태표, 현금흐름표)을 교차 검토한다.
3. 회계정책, 추정, 공시 주석의 변화가 결과에 미치는 영향을 확인한다.
4. 이상 징후, 비정상 변동, 계정 간 불일치를 체크한다.
5. 결론은 사실, 해석, 리스크를 분리해 정리한다.
