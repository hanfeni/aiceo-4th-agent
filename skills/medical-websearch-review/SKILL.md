# medical-websearch-review

## When to use
- 의학 정보에 대해 최신 근거를 확인해야 할 때
- 하나의 출처가 아니라 다수 레퍼런스를 교차 검증해야 할 때
- 가이드라인, 리뷰, 논문, 공신력 있는 기관 자료를 함께 확인해야 할 때
- 질환, 약물, 검사, 치료법의 최신 업데이트가 중요한 질문일 때

## How
1. 질문의 핵심 의학 용어를 표준화하고, 필요한 경우 동의어·영문명을 정리한다.
2. 최신성 우선순위를 적용해 검색한다.
   - 최우선: 최신 임상가이드라인, 공신력 있는 기관/학회 자료
   - 차선: 최근 리뷰 논문, 메타분석, 대규모 임상시험
   - 보조: 교과서적 정의는 배경 확인용으로만 사용
3. 최소 2개 이상의 독립적인 레퍼런스로 동일한 사실을 교차 검증한다.
4. 연도, 발행기관, 문서 종류를 함께 기록해 최신 여부를 판단한다.
5. 상충되는 정보가 있으면 더 최신이거나 근거 수준이 높은 자료를 우선한다.
6. 답변에는 단정적 표현을 줄이고, 확인된 사실과 불확실성을 분리해 제시한다.
7. 의학적 안전 이슈(응급 증상, 약물 이상반응, 금기 등)가 있으면 즉시 주의 문구를 포함한다.
8. 출처는 가능하면 제목, 기관/저자, 연도 수준으로 명시하고, 최종 답변은 사용자 이해 수준에 맞게 간결하게 정리한다.

## 참고 원칙
- 최신성보다 정확성을 우선한다.
- 단일 블로그/커뮤니티 자료는 근거로 사용하지 않는다.
- 개인 맞춤 진단이나 치료 지시는 피하고, 필요 시 의료진 상담을 권고한다.
- 레퍼런스가 부족하면 그 한계를 명확히 밝힌다.
